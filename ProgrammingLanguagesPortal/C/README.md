# 第一部分，C编程实战课程-培训班简介
1. 本课程【C编程实战课程】为【数据科学家系列课程】的课程之一，这一系列课程的目标是培训学员成为高薪高效的数据科学家；
2. 本课程前提：有高中基本数学知识；
3. 课程门户为https://github.com/batermj/data_sciences_campaign , 教学大纲，开发环境，代码，算法，文档，都会通过课程门户进行分享；
4. 本期课程开课日期为2018年3月24日；
5. 本期课程收费750元；
6. 每期课程为十二课，三小时／课，共三十六小时／课程; 
7. 本课程通过C语言进行教学;
8. 开发环境为VIM,EMACS；
